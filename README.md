# Employee_absenteeism_distribution
Employee Absenteeism Prediction Using Logistic Regression

Overview
This project is about predicting if an employee will have low absenteeism. We used a machine learning process that includes preparing the data analyzing it creating new features training a model making predictions and evaluating the results. I have chosen the Logistic Regression as our classification algorithm.

Dataset
Dataset Name: Absenteeism at Work

Source:
https://www.kaggle.com/datasets/tonypriyanka2913/absenteeism-at-work

The dataset has information about employees, like age, education, transportation expense, work load, social habits, body mass index and absenteeism hours.

My model did okay. Here are the results:
 Accuracy: 78.38%
 Precision: 77.78%
 Recall: 73.13%
 F1 Score: 75.38%
The Logistic Regression model worked satisfactorily in classifying employees into high and low absenteeism categories.
